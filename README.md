#1 Setup Project

- `npx create-react-app <project-name>`
- `cd <project-name>`
-  `npm start or npm run start  or npx react-scripts start`
-  auto open browserlocalhost:3000
- การออกจาก browserlocalhost ให้กด control + c ใน terminal vsCode
